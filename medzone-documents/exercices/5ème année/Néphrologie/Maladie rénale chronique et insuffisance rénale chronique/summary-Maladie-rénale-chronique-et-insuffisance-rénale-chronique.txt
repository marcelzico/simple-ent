MALADIE RÉNALE CHRONIQUE ET INSUFFISANCE RÉNALE CHRONIQUE – RÉSUMÉ MÉDICAL

DÉFINITIONS :
- MRC : anomalie rénale (biologique, morphologique, histologique) persistant >3 mois.
- IRC : incapacité des reins à assurer leurs fonctions suite à destruction anatomique irréversible des néphrons, avec DFG <60 ml/min/1,73 m².
- IRCT (stade 5) : DFG <15 ml/min/1,73 m², nécessitant transplantation ou épuration extrarénale.

ÉPIDÉMIOLOGIE :
- 10% population mondiale.
- 2-3 fois plus fréquente chez l'homme.
- Âge moyen : 71 ans (France), 45 ans (Madagascar).
- Étiologies principales : vasculaire et diabétique (+++).

DIAGNOSTIC (6 étapes) :

1. AFFIRMER LA MRC/IRC :
   - Protéinurie/albuminurie.
   - Anomalie sédiment urinaire (hématurie, leucocyturie).
   - Anomalie morphologique (hypotrophie rénale, obstruction).
   - Règle : toute élévation de créatinine = IRA jusqu'à preuve du contraire (stigmates de chronicité).

2. STADIFIER (Classification G et A) :
   - G1 : DFG >90 ; G2 : 60-90 ; G3a : 45-60 ; G3b : 30-45 ; G4 : 15-30 ; G5 : <15 (D=dialysé, T=transplanté).
   - Albuminurie : A1 <30 mg/g ; A2 30-300 mg/g ; A3 >300 mg/g.
   - À partir G3a : consultation néphrologique. À partir G3b : préparation à l'EER.

Formules de calcul :
- CKD-EPI : formule recommandée pour DFG (stadification).
- Cockcroft et Gault : pour prescription médicamenteuse (clairance), utilisable « dans la brousse ».
- MDRD : moins sensible au-delà de 60 ml/min.

3. DÉTERMINER LA NÉPHROPATHIE CAUSALE :
   - Glomérulaire : œdèmes, HTA, hématurie avec hématies déformées, albuminurie >50%.
   - Vasculaire : souffle abdominal, HTA réfractaire, flash OAP, hypokaliémie, asymétrie des tailles rénales.
   - Tubulaire : antécédents lithiases, HBP, tumeurs pelviennes, échographie obstructive.
   - Interstitielle : protéinurie <1g/24h tubulaire, acidose hyperchlorémique, leucocyturie sans germes.
   - Héréditaire : antécédents familiaux, surdité, troubles vue, hématurie (polykystose rénale, syndrome d'Alport).

4. ÉVALUER LES FACTEURS DE PROGRESSION :
   - Contrôle strict PA et protéinurie <0,5g/24h.
   - Équilibre diabète, utilisation BSRAA (IEC/ARA2).
   - Arrêt tabagisme (toxique vasculaire, ↑PA, facteur progression).
   - Restriction protidique modérée.
   - Éliminer toute agression rénale aiguë.

5. ÉVALUER LE RISQUE CARDIOVASCULAIRE :
   - MRC = patients à RCV élevé ; IRC stade 5 = RCV très élevé.
   - Scores : Framingham, RISK score.
   - Facteurs spécifiques IRC : troubles métabolisme phosphocalcique (médiacalcose), hyperhomocystéinémie, anémie, insulino-résistance, toxines urémiques.

6. DÉPISTER LES COMPLICATIONS (si DFG<60) :

A. COMPLICATIONS CARDIOVASCULAIRES :
   - HTA (cause ou conséquence, facteur majeur progression).
   - HVG (HTA + anémie).
   - Péricardite urémique (réversible sous dialyse).

B. TROUBLES MINÉRAUX ET OSSEUX (TMO-MRC) :
   Mécanismes : hyperparathyroïdie secondaire, hypocalcémie tardive, hyperphosphatémie tardive, déficit vitamine D active, acidose.
   Ostéodystrophie rénale (OR) :
   - Ostéite fibreuse (hyperparathyroïdie) : douleur osseuse, fracture, déminéralisation, hyperphosphatémie, hypocalcémie, PTH élevée.
   - Ostéomalacie : stries de Looser-Milkman, diminution 25-OH-vitamine D3.
   - Ostéopathie adynamique (FDR : âge, aluminium, diabète, DPCA) : PTH normale/basse.
   Objectifs : calcémie normale, phosphatémie <1,5 mmol/L, PTH 2-9x normale.

C. ANOMALIES ACIDO-BASIQUES :
   - Acidose métabolique chronique : défaut élimination charge acide + ↓absorption bicarbonates.
   - Conséquences : catabolisme protéique, aggravation maladie osseuse, risque hyperkaliémie.
   - Prévention : carbonatémie >22 mmol/L (bicarbonate Na 3-6g/j, eau de Vichy).

D. ANÉMIE : normochrome normocytaire arégénérative.
   - Traitement : corriger carences (fer, folates, B12), optimaliser stock martial (saturation fer >20%, ferritine >200 ng/ml), érythropoïétine.
   - Cibles : 10 g/dL (jeune sans cardiopathie) ; 11-12 g/dL (cardiopathe, diabétique, âgé).
   - Transfusion sanguine à éviter sauf urgence.

E. ANOMALIES HÉMOSTASE :
   - Tendance thrombotique (hyperhomocystéinémie, hyperfibrinogénémie, ↑facteurs VIII/Willebrand).
   - Tendance hémorragique (syndrome urémique : urée diffuse → gingivorragie, hémorragie digestive/génitale).

F. DÉFICIT IMMUNITAIRE : baisse réponse vaccinale.

G. TROUBLES HYDROÉLECTROLYTIQUES :
   - Rétention hydrosodée (OMI, anasarque, OAP, hyponatrémie dilution).
   - Hyperkaliémie (défaut élimination rénale, favorisée par acidose, IEC/ARA2, diurétiques épargneurs K). Urgence si K>7 mmol/L (dialyse). Résines : Resikali, Kayexalate.

H. AUTRES CONSÉQUENCES :
   - Hyperuricémie (asymptomatique, traiter seulement si crise goutte).
   - Hyperlipidémie (hypertriglycéridémie + ↓HDL ± hypercholestérolémie).
   - Dénutrition protéino-énergétique (nausées, vomissements, gastrite).
   - Troubles neurologiques : polynévrites urémiques, crampes, encéphalopathie urémique/hypertensive.
   - Troubles hormonaux : impuissance/aménorrhée, fertilité diminuée.

TRAITEMENT :

Objectifs : maintenir fonction rénale résiduelle, ralentir progression vers stade terminal.

Moyens non médicamenteux :
- Restriction protidique modérée, régime hyposodé.
- Hydratation adaptée, activité physique régulière.

Moyens médicamenteux :
- BSRAA (IEC/ARA2) : néphroprotecteurs. Initier à faible dose (¼ cp le soir), augmentation progressive. Si élévation créatinine >15% de la baseline → diminuer la dose.

Traitement IRCT (stade G5) :
- Suppléance (EER) : hémodialyse, dialyse péritonéale (ne remplace pas fonctions endocriniennes).
- Remplacement : greffe rénale (remplace fonctions endocriniennes).

Traitement étiologique : diabète, lupus, etc.

EN SOMME – DÉMARCHE PRATIQUE :
1. MRC = anomalie rénale >3 mois (sang, sédiment urinaire).
2. Évolution vers IRC = baisse progressive et irréversible DFG <60 ml/min + complications.
3. Orientation vers suivi spécialisé.
